# portfolio

This is a small static portfolio site for Alwin Siemens (data scientist).

Preview locally

Open a terminal in the repository root and run a simple static server. If you have Python installed:

```powershell
# For Python 3
python -m http.server 8000

# then open http://localhost:8000 in your browser
```

Files

- `index.html` — single-page site
- `assets/styles.css` — styles
- `assets/script.js` — small interactive behaviors

Notes

- This is a static starter site. Replace placeholder project descriptions and images with your own content.
- To accept contact messages, wire the form in `index.html` to an email service or backend endpoint.
